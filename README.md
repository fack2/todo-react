# todo-react
To Do App, for the ReactJs week.

## User Journey
 - As a user I'm able to add task.
 - As a user I'm able to delete tasks.

## File Structure
 - public
   - index.html
 - src
   - Component
     - ItemList.js
   - App.js
   - index.js
   - index.css
 - README.md
 - .gitignore
 - package.json
 
 ## How to run project?
 
   - **git clone** this repo.
   - **npm i**.
   - **npm start**.
