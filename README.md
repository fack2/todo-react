# todo-react
To Do web application, for the React week.

## User Journey
 - As a user I'm able to add task.
 - As a user I'm able to delete tasks.

## File Structure
 - public
   - index.html
   - style.css
 - src
   - components
     - AddTodoItem.js
     - TodoList.js
   - App.js
   - index.js
 - README.md
 - .gitignore
 - package.json
 
 ## How to run project?
   - **git clone** this repo.
   - **npm i**.
   - **run start**.
